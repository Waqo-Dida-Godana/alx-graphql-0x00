# Episode Queries

This directory contains GraphQL queries for fetching episode data from the Rick and Morty API.

## Files

- `episode-page-1.graphql` - Query to fetch a specific episode by ID
- `characters-page-1-output.json` through `characters-page-4-output.json` - Output files for character queries
- `characters-page-1.graphql` through `characters-page-4.graphql` - Character query files

## API Endpoint

All queries are executed against: `https://rickandmortyapi.com/graphql`
